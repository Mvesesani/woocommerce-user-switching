Extend 'User Switching' to add functionality for WooCommerce.

== Description ==

This plugin is an extension to the "User Switching" plugin.
It adds links to the "Orders" and "Order Detail" pages of WooCommerce that will allow admins to quickly switch to customer profiles.

== Installation ==

1. Ensure that you first have the `WooCommerce` plugin installed.
2. Ensure that you secondly have the `User Switching` plugin installed.
3. Upload the `woocommerce-user-switching` folder to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I know that it is working? =

Check the WooCommerce 'Orders' page, you should see a new column titled "Switch to User".
